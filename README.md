# sputnik++

> base de cliente de inyección JNI para Minecraft Java 1.21.11

## por qué

para que skids como yo puedan crear su propio cliente JNI. y para aprender C++.

## cómo skiddear

- `git clone https://github.com/retuci0/sputnikplusplus`

- usar IDE preferido (yo uso VS2022) para añadir lo que sea que se vaya a añadir

## cómo inyectar

de momento, hasta que haga un inyector, usar Process Hacker 2 o alguna utilidad por el estilo para inyectar el DLL
